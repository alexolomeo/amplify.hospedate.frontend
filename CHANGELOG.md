## [1.0.1] - 2025-10-09

### Added

- Automated versioning with standard-version
- Changelog generation based on conventional commits
- Enhanced error handling and logging in authentication and token management
- Detailed logging for Google OAuth token login failures
- Token cleanup mechanisms to prevent reuse of consumed tokens
- Visibility change handler for proactive session refresh after computer sleep
- Selected listings persistence feature
- Suggestions and select filters per status

### Fixed

- Duplicate Google OAuth token login API calls
- Implemented global guard in loginWithToken service
- Converted usePopupListener to singleton pattern
- Session loss issues due to overly aggressive error handling in `isInvalidRefresh()`
- Token reuse causing 400 Bad Request errors on Google login
- Stale token cleanup after 5 minutes to prevent localStorage pollution
- Profile and card message issues

### Changed

- Removed outdated release documentation and quickstart guides
- Improved `isInvalidRefresh()` to only clear session on actual authentication failures (401 with specific codes)
- Enhanced duplicate token prevention in `usePopupListener`
